# Verbum Error Taxonomy

The taxonomy groups validated rule-backed errors so the current submission is easier to read and drill.

It is not a long-term analytics system in V0.

## Categories

- `AGREEMENT`: gender and number concordance.
- `VERB_SYSTEM`: ser/estar, tense, mood, and verb constructions.
- `PRONOUN`: clitic order, object pronouns, and reflexive forms.
- `PREPOSITION`: por/para, a personal, direction, and verb frames.
- `LEXICAL_GENDER`: noun gender exceptions.
- `USAGE`: curated false cognates and collocations.
- `OTHER`: fallback for validated rules without category metadata.

## Use In V0

- Show a compact breakdown for the current diagnosis.
- Help choose drills from the current errors.
- Support rewrite comparison within the current browser session.

## Not In V0

- Historical dashboards.
- Recurrence tracking across days.
- Mastery scores.
- Persistence.
